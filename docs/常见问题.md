# 常见问题

## 登录问题

如果遇到登录问题：
1. 检查邮箱配置是否正确
2. 查看后端日志
3. 清理旧的验证码记录：`npx tsx scripts/cleanup-verification-codes.js`

## 数据库问题

- 迁移失败：确保 MySQL 服务已启动
- 连接错误：检查 `.env` 中的 `DATABASE_URL`

## 服务器重启

Windows PowerShell:
```powershell
taskkill /F /IM node.exe
Start-Sleep -Seconds 2
npm run dev
```
